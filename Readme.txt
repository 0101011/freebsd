# $FreeBSD$
This source tree is no longer being maintained.

The pNFS server code is now all in head/current and will be patched
there, but not here.

If you want to run the pNFS server, you should use head/current, which
will be released as FreeBSD-12.
I'd suggest the most recent FreeBSD-12 snapshot/Beta/RC before 12.0
is actually released.
